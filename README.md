# CrackLS315

## Overview

CrackLS315 is a task-specific segmentation package derived from the DeepCrack dataset release.

- Task: segmentation
- Images: 315
- Annotations: 855
- Classes: 1 (crack)
- Annotation format: paired bitmap masks

The dataset contains line-array pavement images captured under laser illumination.

- Name: CrackLS315
